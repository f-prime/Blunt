Blunt

A CSS layout framework that gets to the point and stays out of your way.

Star

Blunt is designed to make creating page layouts a easy. It provides a host of general CSS classes to position elements, add padding and margins to elements, create responsive grids, and more. It has no opinions on how your page should look. This means that you won't have to fight with Blunt to overwrite default colors and button styles. All of that is left up to you.

You can find Blunt on GitHub and join the discussion on Gitter.

Getting Started

Include Blunt into your project with the this single line of code.

<link rel="stylesheet" type="text/css" href="https://cdn.jsdelivr.net/gh/f-prime/blunt/src/blunt.css">

Basics

With the exception of the container class, all classes in Blunt follow the same structure.

{size}-{className}-{value}

Where {size} is replaced with a screen size lg, md, or sm, {className} is replaced with the specific Blunt class and {value} is replaced with a number.

Blunt currently supports three screen sizes sm (max width 900px) md (max width 1500px) and lg (min width 1500px)

As a practical example, let's say we wanted to add padding to the top of an element.
If we wanted to do this for all screen sizes we would do this.

<div class="pt-1">

If we wanted to have this padding on only spall screens (screen size < 900px) we would do the following

<div class="sm-pt-1">

The padding class along with some other classes also support decimal values using the notation

{number}p{fraction}

So if we wanted to add, say, a padding of 1.3 to the top of an element we could do the following.

<div class="pt-1p3">

Container

The container class creates an element with responsive width.

<div class="container"> Other stuff </div>
Width and Height

Blunt provides 3 different width and height classes

{h,w}-{1-100}, px{h,w}-{1-1000} and v{h,w}-{1-100}

<div class="w-50">

Makes the div's width 50%

<div class="vh-50">

Makes the div's height 50vh

<div class="md-pxw-500">

Makes the div's width 500px on medium sized screens

Margin and Padding

The following padding and margin classes are provided

{m,p}{l,r,t,b}-{0-15}p{1-9}

<div class="mt-5">

Adds a top margin of 5vh to the div

<div class="pl-3">

Adds a padding left of 3vw to the div

<div class="sm-mb-2p5">

Adds a margin of 2.5vh to the div on small screens

<div class="md-pr-10">

Adds a padding right of 10vw to the div on medium screens
